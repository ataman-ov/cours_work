# Ситуационный центр

Учебное desktop-приложение на C# WinForms для мониторинга показателей виртуального датчика.

## Что умеет приложение

- получает показания от виртуального датчика температуры;
- сохраняет каждое показание в файловую базу данных JSON;
- показывает историю показаний в интерфейсе;
- проверяет норму: от 18 до 32 °C;
- если показатель вне нормы, автоматически создает задачу виртуальному сотруднику;
- хранит список сотрудников с именем, должностью, подразделением и телефоном;
- позволяет отметить задачу выполненной.

## Структура проекта

- `SituationCenter.csproj` - проект C# WinForms;
- `Program.cs` - точка входа;
- `MainForm.cs` - интерфейс и основная логика;
- `VirtualSensor.cs` - виртуальный датчик;
- `FileDatabase.cs` - сохранение и загрузка файловой БД;
- `Models.cs` - модели данных;
- `data/situation-center-db.json` - файловая база данных.

## Как запустить

1. Откройте папку `SituationCenter` в Visual Studio 2022.
2. Убедитесь, что установлен .NET 8 SDK и компонент разработки desktop-приложений .NET.
3. Запустите проект кнопкой `Start` или командой:

```powershell
dotnet run --project SituationCenter.csproj
```

При запуске приложение использует файл `data/situation-center-db.json` рядом с исполняемым файлом. Если файла нет, он будет создан автоматически с начальными сотрудниками.
